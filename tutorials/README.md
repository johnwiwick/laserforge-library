# LaserForge Tutorials

Drop tutorial video files into this folder structure and LaserForge will turn them into menu items automatically.

## Supported file types

- `.mp4`
- `.mov`
- `.m4v`
- `.webm`

## Folder structure

- `tutorials/getting-started/`
- `tutorials/box-and-shape-builds/`
- `tutorials/surface-design/`
- `tutorials/clock-tools/`

File names become menu labels automatically.

Example:

- `finger-joint-setup.mp4` becomes `Finger Joint Setup`
- `using-the-design-library.mov` becomes `Using The Design Library`
