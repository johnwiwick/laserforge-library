# LaserForge Library

Public design library for John Wick LaserForge.

This repository feeds the in-app Design Library window. Keep artwork simple, high-contrast, and engraving-friendly.

## Folder layout

- `borders/` contains frame-style artwork and edge treatments.
- `patterns/` contains repeatable fills, tiles, and decorative line art.

## Recommended artwork rules

- Prefer black line art on a transparent or white background.
- SVG is best for crisp scaling.
- PNG and JPG are allowed for artwork that should be dithered.
- Keep filenames short and descriptive.
